Projeto de final de curso de universidade

Website completo para um órgão de comunicação social, utilizando arquitetura MVC. Implementação de frontend com HTML, CSS e JavaScript e backend com Django, incluindo gestão de base de dados e integração com MySQL. O sistema inclui autenticação de utilizadores, área exclusiva para subscritores, comentários restritos, chat em tempo real e um módulo de submissão de notícias com validação baseada em inteligência artificial.
